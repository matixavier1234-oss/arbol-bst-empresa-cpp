# Árbol BST Empresarial en C++

## Integrantes
- Mena Romero Matías Xavier

## Objetivo
Implementar un árbol binario de búsqueda en C++ para organizar empleados de una empresa.

## Funcionalidades
- Insertar empleados
- Buscar empleados
- Mostrar raíz
- Recorridos inorden, preorden y postorden
- Calcular altura
- Mostrar nodos hoja

## Teoría
- **Raíz:** Es el nodo principal y el punto de partida de toda la estructura. Es el único nodo en todo el árbol que no tiene un nodo "padre" por encima de él. Representa la cabeza de la organización (el código 50, "Empresa UTA").
- **Nodo Interno:** Es cualquier nodo que tiene al menos un nodo "hijo" conectado por debajo de él. Tienen un padre (excepto la raíz) y descendientes.
- **Hoja (Nodo Hoja):** Son los nodos que se encuentran en los extremos del árbol. Su característica principal es que no tienen ningún hijo.
- **Nivel:** Representa la profundidad o la "generación" en la que se encuentra un nodo. La raíz siempre se encuentra en el Nivel 0. 
- **Altura:** Es la distancia máxima (el número de niveles o conexiones) que existe desde la raíz hasta la hoja más lejana.
